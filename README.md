# TaskFlow SaaS - Frontend 🚀

A premium Multi-User Task Management System frontend built with **React**, **Vite**, and **Tailwind CSS**. This application features a modern Glassmorphism UI and a seamless user experience.

## ✨ Features
- **Modern UI**: Clean, responsive Glassmorphism design using Tailwind CSS.
- **Secure Authentication**: Integrated with JWT-based backend authentication.
- **Task Management**: Full CRUD operations for personal tasks.
- **State Management**: Efficient use of React Hooks for real-time UI updates.
- **Responsive**: Fully optimized for mobile, tablet, and desktop screens.

## 🛠 Technologies
- **React**: Component-based UI development.
- **Vite**: Ultra-fast build tool and development server.
- **Tailwind CSS**: Modern utility-first styling with custom glassmorphism effects.
- **Lucide Icons**: Beautiful, consistent iconography.
- **Axios**: Promised-based HTTP client for API requests.
- **React Router**: Seamless navigation and protected routes.

## 🚀 Deployment (Vercel)
This frontend is ready to be deployed on **Vercel**.
1. Connect this repository to Vercel.
2. Set the Environment Variable:
   - `VITE_API_URL`: Your backend API URL (e.g., `https://your-backend.up.railway.app/api`).
3. Deploy!

## 💻 Local Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
